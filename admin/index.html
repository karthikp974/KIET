Admin sign-in

You signed in with the view-only password. Tabs for editing the homepage, programs, and Save website stay hidden. Use your full admin password to upload images, add rows, and save changes.

Uploads from PC or phone: choose a file in admin — the server saves it, converts photos to JPEG and compresses toward about 100KB (resize + quality) for faster pages, with a correct .jpg URL. Railway: set env UPLOADS_DIR to a folder on a persistent volume (e.g. /data/uploads) so files survive redeploys. Otherwise use full https://… image links for anything you cannot afford to lose.
